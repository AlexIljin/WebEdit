Release checklist.

- Update Changelog.
- Update WebEdit.txt (copyright dates, feature list).
- Update Version Information (ObideVer, ObideUVer, ObideVer.rct, ObideUVer.rct).
- Update WebEdit/WebEditU.AboutMsg text.
- Update copyright years in the sources.
- Tag the release using tag.bat.
- File release in Notepad++ Plugins project:
   https://sourceforge.net/project/admin/explorer.php?group_id=189927
- Update Plugin Manager's plugin list.
- Drop note in Notepad++ Plugin Development forum:
   https://sourceforge.net/forum/forum.php?thread_id=1959856&forum_id=482781
- Update PluginCentral Wiki page:
   https://sourceforge.net/apps/mediawiki/notepad-plus/index.php?title=Plugin_Central
